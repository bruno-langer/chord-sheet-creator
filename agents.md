# Agents Documentation

## Overview

This project uses AI agents to help with various tasks related to song configuration and management. Agents are specialized AI assistants that can perform specific functions to streamline your workflow.

## Available Agents

### Song Configuration Agent
- **Purpose**: Helps configure and manage song sections, chords, and structure
- **Capabilities**:
  - Add/remove song sections
  - Modify chord progressions
  - Adjust repeat counts
  - Preview song layouts

### Export Agent
- **Purpose**: Handles exporting songs to various formats
- **Capabilities**:
  - Generate HTML viewers
  - Create downloadable files
  - Format output for different platforms

### Preview Agent
- **Purpose**: Provides real-time preview of song configurations
- **Capabilities**:
  - Live updates as you edit
  - Visual representation of sections
  - Chord display formatting

## How to Use Agents

1. **Select an Agent**: Choose the appropriate agent for your task from the available options
2. **Provide Instructions**: Give clear, specific instructions about what you want to accomplish
3. **Review Output**: Check the agent's work and provide feedback if needed
4. **Iterate**: Make adjustments and refinements as necessary

## Best Practices

- Be specific in your requests
- Provide context when possible
- Test outputs thoroughly
- Save your work regularly

## Troubleshooting

If an agent doesn't perform as expected:
1. Check your instructions for clarity
2. Provide more context or examples
3. Try breaking complex tasks into smaller steps
4. Contact support if issues persist
